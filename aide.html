❓ Aide - Application ScanProf

ScanProf est une application pensée pour les professeurs d'EPS afin de collecter, trier, organiser et exploiter les données scannées ou importées depuis d'autres applications (comme RunStats).

📷 Page d'accueil (Scan)

  • Utilisez le scanner QR Code pour enregistrer les élèves. Il ouvre automatiquement la caméra arrière (si disponible).
  • Les élèves sont automatiquement enregistrés (si non déjà présents) et redirigés vers la page des participants.
  • Vous pouvez également importer un fichier CSV contenant des élèves déjà scannés ou issus d’une précédente séance.

👥 Participants enregistrés

  • Liste des élèves enregistrés (après scan ou import CSV).
  • Vous pouvez trier dynamiquement selon n’importe quelle colonne du tableau (ex : Nom, Classe, VMA, Distance...)
  • Recherche rapide par nom, prénom ou classe (ex : taper "5A" ou "Lola").
  • Boutons disponibles :
    • 🏠 Retour à l'accueil
    • ⚙️ Tri dynamique (choix d'une colonne)
    • 🔍 Recherche texte
    • ⬇️ Exporter CSV (sauvegarde localement le fichier, ex. dans Fichiers sur iPad)
    • 📥 Importer un fichier CSV
    • 🧬 Groupe ZENOS Tour (redirige vers la page des groupes)
    • 🧼 Réinitialiser (efface toutes les données)

🧬 Groupes ZENOS Tour

  • Fonction dédiée à l’événement ZENOS Tour
  • Forme des groupes de 4 élèves mixtes :
    • 1 VMA haute
    • 1 VMA basse
    • 2 VMA moyennes
  • Les élèves restants (non groupables par 4) ne sont pas affichés
  • Boutons disponibles :
    • 🏠 Accueil
    • 🖶 Imprimer
    • 📄 Générer PDF (tous les groupes sur une seule page)
    • ❓ Aide

📁 Fichiers CSV

  • L’export CSV permet de sauvegarder les données dans l’iPad (via “Fichiers”).
  • L’import CSV permet de réutiliser un fichier déjà enregistré (utile pour poursuivre une séance ou créer de nouveaux groupes).

🔒 Données

  • Les données sont stockées temporairement dans le navigateur (localStorage).
  • Une réinitialisation efface tout (pensez à exporter avant).
