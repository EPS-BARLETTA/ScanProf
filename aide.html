📘 Aide - ScanProf

1. Accueil

  • Scanner un QR code élève (depuis RunStats ou autre).
  • Chaque élève est automatiquement ajouté au stockage local.
  • Un message vert confirme l’ajout (ou orange si déjà scanné).
  • Vous pouvez ensuite cliquer sur “Voir les participants”.

2. Page “Participants enregistrés”

  • Liste de tous les élèves scannés ou importés.
  • Tri dynamique possible selon n’importe quelle colonne (Nom, Classe, Sexe, VMA, Distance...)
  • Recherche texte : tapez “Lola” ou “5A” pour filtrer la liste.
  • Exporter CSV : crée un fichier avec tous les élèves.
  • Importer CSV : recharge une ancienne liste (scannée auparavant ou enregistrée dans Fichiers iPad).
  • Réinitialiser : efface tous les élèves stockés.
  • “Groupe ZENOS Tour” : envoie vers la page de création de groupes.

3. Page “Groupes ZENOS Tour”

  • Crée des groupes mixtes de 4 élèves selon la VMA.
  • Chaque groupe : 1 VMA haute, 1 basse, 2 moyennes, mixité assurée.
  • Seuls les groupes complets sont affichés.
  • Possibilité d’imprimer ou de générer un PDF avec tous les groupes.

4. Fonctionnement sur iPad

  • Compatible avec Safari (et Chrome si autorisé à accéder à la caméra).
  • Le bouton “Exporter CSV” permet d’enregistrer les données dans “Fichiers”.
  • Pour importer un fichier : ouvrez à nouveau la page “Participants”, cliquez sur “Importer” et sélectionnez un fichier CSV déjà existant.
