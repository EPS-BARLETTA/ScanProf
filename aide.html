📚 Aide - Application ScanProf

Dernière mise à jour : Août 2025

🔎 À quoi sert ScanProf ?

ScanProf est une application Web conçue pour les professeurs d'EPS. Elle permet de :

  • Scanner ou importer des données élèves issues d’autres applications EPS (ex. RunStats, tournoi, escalade...)
  • Afficher les élèves automatiquement dans un tableau
  • Trier dynamiquement selon les colonnes détectées (nom, classe, VMA, distance, points, etc.)
  • Exporter les données en CSV ou imprimer les résultats
  • Générer des groupes ZENOS Tour hétérogènes (spécial RunStats)
  • Créer un PDF récapitulatif des groupes ZENOS

📥 1. Scanner ou importer un fichier

Depuis la page d’accueil, vous pouvez :

  • Scanner un QR code généré par une autre application (ex. RunStats)
  • Ou importer un fichier .csv contenant les données élèves

📊 2. Résumé des élèves

Les élèves sont affichés automatiquement dans un tableau :

  • Un menu déroulant permet de trier par nom, classe, sexe, VMA, etc.
  • Une zone de recherche permet de filtrer par mots-clés (ex. “5A”, “Lola”)
  • Vous pouvez exporter les données en CSV ou imprimer le tableau
  • Bouton “Groupe ZENOS Tour” si vous utilisez un fichier RunStats compatible

🧬 3. Groupes ZENOS Tour (spécifique RunStats)

Ce module crée des groupes hétérogènes de 4 élèves selon la VMA et la mixité :

  • 1 élève VMA basse + 2 élèves VMA moyenne + 1 élève VMA haute
  • Groupes obligatoirement mixtes
  • Les élèves non répartis (si groupe incomplet ou non mixte) sont affichés à part

📄 4. Générer un PDF

Un bouton dédié vous permet de générer un PDF :

  • Affiche tous les groupes ZENOS sur une seule page
  • Colonnes : Nom, Prénom, Distance, VMA
  • Titre : ZENOS TOUR
  • Pied de page : ScanProf - Équipe EPS Lycée Vauban - LUXEMBOURG - JB

📈 5. Analyse visuelle (graphique)

Un bouton orange permet d'afficher un graphique clair :

  • Répartition des élèves par tranches de VMA et Distance
  • Présentation sous forme d’histogrammes colorés
  • Permet de visualiser l’équilibre des groupes

💡 Astuces et idées d’évolution

  • Les fichiers CSV exportés peuvent être stockés sur iPad mais ne sont pas réutilisables automatiquement dans l’application.
  • À l’avenir, une fonction d’import de résultats d’une séance précédente pourrait être ajoutée (pour suivi, progression, etc.)
